## 0.1.0

- Initial developer-preview Flutter SDK for KittenTTS.
- Supports Android, iOS, macOS, Linux, Windows, and Flutter Web builds with ONNX Runtime.
- Uses the CE phonemizer on Flutter native targets and the same JS/Emscripten CE runtime style as the React Native web package on Flutter Web.
- Includes runtime model downloads, local cache reuse, built-in voices, WAV output, word timings, streaming generation, and example apps.
